settings for filming on Canon DSLR


These are the setting suggested by vincent Laforet, to obtain a flat image to work with in post-production
  1. Shootin in Manual
  2. Set Live view to movie display
  3. Record at 1920x1080 24fps
  4. Disable highlight tone priority
  5. Use the following ISO 160/320/640/1250
  6. Set shutter at 50 (and leave it there)
  7. Set pitcture style to
    1. Nenutral
      1. Sharpness 0
      2. Contrast - 4
      3. Saturation - 2
    2. Or use Cinestyle with 
      1. Sharpness 0
      2. Contrast -4
      3. Saturation - 2
